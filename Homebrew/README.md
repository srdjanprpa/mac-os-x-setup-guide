# Instalacija

Pokrenite instalaciju preko komande linije

        $ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Da bi proverili da li je Homebrew instaliran, otkucajte u terminalu sledece

        brew help
